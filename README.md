# midtearm-python
This is a simple Recipe Manager program written in Python.
The program allows the user to store, view, search, and manage recipes using a CSV file.

This project was created as a first programming project to practice:

lists

dictionaries

functions

loops

file handling with CSV

basic menu systems

Features

The program allows the user to:

Add a recipe

Name

Cuisine

Difficulty

Preparation time

Ingredients

Category

View all recipes

Recipes are shown in alphabetical order by name.

Search recipes

Search by cuisine, category, or ingredient.

Delete a recipe

Delete a recipe using its ID.

View statistics

Total number of recipes

Recipes grouped by cuisine

Recipes grouped by ingredient

Recommend a recipe

User chooses a category

Program randomly recommends a recipe from that category.

Save and exit

All recipes are saved to a CSV file.

Files

recipes.py → main Python program

recipes.csv → stores all recipes

README.txt → project information
